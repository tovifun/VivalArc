# VivalArc - Compact UI Variant

⚠️ **兼容性保留版本** - 后续优化以根目录 `VivalArc/` 为主

## 📍 目录选择

在 Vivaldi 设置中选择此目录：`VivalArc/variants/compact/`

## 📦 版本信息

- **VivalArc 版本**: v1.2.0
- **更新日期**: 2025-11-30
- **支持 Vivaldi**: v7.7+
- **维护状态**: 仅保留，不再单独迭代

## ✨ 特性

- **紧凑布局**: 减小 UI 元素间距和尺寸
- **更多内容空间**: 在小屏幕上显示更多内容
- **精简设计**: 适合追求效率的用户

## 💡 适用场景

- 小屏幕设备（笔记本、平板等）
- 需要同时查看多个窗口的用户
- 喜欢紧凑界面风格的用户
