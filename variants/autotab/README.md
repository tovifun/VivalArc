# VivalArc - Auto Hide Tabbar Variant

⚠️ **兼容性保留版本** - 后续优化以根目录 `VivalArc/` 为主

## 📍 目录选择

在 Vivaldi 设置中选择此目录：`VivalArc/variants/autotab/`

## 📦 版本信息

- **VivalArc 版本**: v1.2.0
- **更新日期**: 2025-11-30
- **支持 Vivaldi**: v7.7+
- **维护状态**: 仅保留，不再单独迭代

## ✨ 特性

- **自动隐藏标签栏**: 鼠标移开时自动隐藏标签栏，鼠标悬停时显示
- **节省屏幕空间**: 提供更大的浏览区域
- **保留所有功能**: 标签栏功能完整保留，仅视觉上隐藏

## 💡 适用场景

- 喜欢极简界面的用户
- 需要更大浏览区域的场景
- 经常使用快捷键管理标签的用户
