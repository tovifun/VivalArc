# VivalArc 历史变体说明

仓库根目录下的 `vivalarc.css` 现在是 VivalArc 唯一持续维护的版本。

## 当前维护策略

`variants/` 目录下的样式会继续保留，主要作为历史兼容配置：

- `variants/autotab/`
- `variants/compact/`

但后续修复和优化会优先收敛到根目录版本，不再继续扩展多套变体分支。

## 建议使用方式

如果你是首次使用，或准备跟随后续版本升级，请在 Vivaldi 设置 > 外观 > 自定义 UI MOD 中直接选择 `VivalArc/`。

如果你目前仍在使用历史变体，遇到兼容性问题时，建议先切回根目录版本确认问题是否已经解决。后续新增优化默认不会再同步维护到独立变体目录。
