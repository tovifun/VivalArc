# VivalArc for Vivaldi 6.9

⚠️ **此版本已归档** - 建议升级到最新版本

## 📍 目录选择

在 Vivaldi 设置中选择此目录：`VivalArc/archive/v6.9/default/`

## 📦 版本信息

- **VivalArc 版本**: v1.0.4
- **更新日期**: 2024-08-17
- **支持 Vivaldi**: v6.9.x

## ⚠️ 已知限制

- 仅支持 Vivaldi 6.9 版本
- 部分新特性不可用
- 不再接受功能更新
- 后续维护以根目录版本为主

## 📝 版本更新日志

### v1.0.4 (2024-08-17)
**针对 Vivaldi 6.9 的优化**
- 修复侧边栏背景多出的半透明层问题 [#24](https://github.com/tovifun/VivalArc/issues/24)
- 新增多个 VivalArc 变体供选择
- 推出 Reflect New Tab 渐变主题系列

## 🔼 升级指南

### 升级到最新版本 (v1.3.0)

1. **备份当前配置**（如果有自定义修改）

2. **在 Vivaldi 设置中更改目录**
   - 推荐：选择根目录 `VivalArc/`（v7.9+ 默认版本）

3. **重启 Vivaldi**

### 注意事项

- 升级后某些 CSS 变量名称可能有变化
- 建议先在测试环境试用新版本
- 如遇问题可随时切换回此版本
