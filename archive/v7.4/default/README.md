# VivalArc for Vivaldi 7.4

⚠️ **此版本已归档** - 建议升级到最新版本

## 📍 目录选择

在 Vivaldi 设置中选择此目录：`VivalArc/archive/v7.4/default/`

## 📦 版本信息

- **VivalArc 版本**: v1.1.0
- **更新日期**: 2025-05-15
- **支持 Vivaldi**: v7.4

## ⚠️ 已知限制

- 仅针对 Vivaldi 7.4 优化
- 较新版本可能有兼容性问题
- 不再接受功能更新
- 后续维护以根目录版本为主

## 📝 版本特性

此版本是基于 v7.4 的优化版本，包含：
- 针对 Vivaldi 7.4 的样式调整
- 优化的窗口边框默认值（3px）
- 改进的标题栏高度计算

## 🔼 升级指南

### 升级到最新版本 (v1.3.0)

**推荐选择：** `VivalArc/`（根目录，v7.9+ 默认版本）

**步骤：**
1. 确保 Vivaldi 已升级到 v7.9 或更高版本
2. 在 Vivaldi 设置中更改 CSS MOD 目录为 `VivalArc/`
3. 重启 Vivaldi
