# VivalArc for Vivaldi 7.0

⚠️ **此版本已归档** - 建议升级到最新版本

## 📍 目录选择

在 Vivaldi 设置中选择此目录：`VivalArc/archive/v7.0/default/`

## 📦 版本信息

- **VivalArc 版本**: v1.0.5
- **更新日期**: 2024-11-03
- **支持 Vivaldi**: v6.6 - v7.0

## ⚠️ 已知限制

- 不支持 Vivaldi 7.1 及更高版本的新特性
- 不再接受功能更新和优化
- 后续维护以根目录版本为主

## 📝 版本更新日志

### v1.1.0 (2024-11-03)
**Vivaldi 7.0 适配与功能增强**

Vivaldi 7.0 带来了显著的 UI 改进：
- 更精致的图标
- 更现代的界面设计
- 更细腻的侧边栏选中标签投影

**新增功能：**
1. Footer 现在默认可见（可在设置中隐藏）[#28](https://github.com/tovifun/VivalArc/issues/28)
2. 弹出页面地址栏支持拖拽 [@Zettry 贡献] [#30](https://github.com/tovifun/VivalArc/issues/30)
3. 恢复标签栏滚动条显示（解决宽度调整问题）[#31](https://github.com/tovifun/VivalArc/issues/31)
4. 新增自动隐藏标签栏变体 [#21](https://github.com/tovifun/VivalArc/issues/21), [#29](https://github.com/tovifun/VivalArc/issues/29)

## 🔼 升级指南

### 升级到最新版本 (v1.3.0)

1. **检查 Vivaldi 版本**
   - 确保已升级到 Vivaldi 7.9 或更高版本

2. **更改目录**
   - 选择根目录 `VivalArc/`

3. **重启 Vivaldi**
